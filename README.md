*Bookit* - Your Personal Library

Bookit is a comprehensive web application designed for book enthusiasts to discover, organize, and review their favorite reads. From managing a personal bookmark list to exploring over a million titles across various genres, Bookit serves as a digital companion for your reading journey.

*Tech Stack*

Frontend:

HTML5: Structure of the library and user forms.

CSS3: Custom styling (including the leaf-patterned background and card-based layouts).

JavaScript: Client-side interactions, such as the "Book bookmarked!" alerts and star-rating UI.

Backend:

PHP: Server-side logic for handling user sessions, database queries, and reviews.

Database:

MySQL: Storing user credentials, book metadata, reviews, and bookmark relations.

Server Environment:

XAMPP: Local development environment using Apache as the web server.
